title: "Predicting the Elastic Modulus of Steel Fiber Reinforced Concrete using Machine Learning"
excerpt: "This ongoing study develops and benchmarks machine learning models to predict the elastic modulus (E, MPa) of Steel Fiber Reinforced Concrete (SFRC) using a compiled database of 233 specimens from 20 published sources (1993–2025). Input features include compressive strength (f'cf), fiber volume fraction (Vf%), aspect ratio (l/d), water-to-cement ratio (W/C), and water-to-binder ratio W/(C+S). Models evaluated so far include Linear Regression, Decision Tree, and Random Forest, with Random Forest achieving R²=0.765 on the test set — substantially outperforming the best existing empirical formula (ACI 363R, R²=0.216). Further ensemble and kernel-based methods (Extra Trees, Gradient Boosting, SVR, KNN) are currently under investigation. The study also introduces two engineered features — a reinforcement index (RI) and an SF percentage feature — and rigorously validates all models using 5-fold cross-validation on the training set only to prevent data leakage."
